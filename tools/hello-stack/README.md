# Hello-Stack Templates

This folder contains minimal templates to bootstrap:

- OpenCode MCP wiring for Serena + retrieval MCP servers
- OpenAgentsControl registry/profile intent for one end-to-end path

Files:

- `opencode.mcp.example.json`: MCP server layout for OpenCode
- `oac.registry.example.json`: OAC-style orchestration shape for one task flow
- `quickstart.md`: practical bootstrap and smoke workflow
- `check_stack.py`: local environment and config validation checks

Notes:

- These are starter templates, not drop-in production config.
- Validate against the exact OAC schema/version in your vendor snapshot.
- Review jCodeMunch/jDocMunch licensing terms before commercial use.
