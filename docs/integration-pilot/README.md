# Integration Pilot Documentation Guide

Last reviewed: 2026-04-11
Primary evidence window: 2026-03-22 pilot cycle

This folder is retained as pilot evidence for the OpenCode integration rollout.

## Status Classification

- Default classification for `docs/integration-pilot/*`: **Historical evidence**
- Use as reference material for:
  - validated pilot commands and outputs
  - guardrails, rollback drills, and decision records
  - assumptions and failure modes observed during pilot

## How to Use This Folder

- Do not treat these files as the default source of current production procedures.
- When consuming a pilot file operationally, verify command paths and runtime assumptions first.
- Prefer current docs indexes for active guidance:
  - `docs/README.md`
  - `docs/oc/README.md`
  - `docs/governance-transition.md`
